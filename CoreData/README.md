# Core Data


Implement the BuzzwordCoreDataStore. Before you beginn implementing code, create a xcdatamodel with an entity "BuzzwordEntity" and the field: id, name, count. Don't forget to generate the corresponding NSManagedObject Subclass. Use the Unit Tests to guide you through the implementation (CoreDataStackTest.swift and BuzzwordCoreDataStoreTest.swift)
